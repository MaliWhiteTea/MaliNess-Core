## Action referansı (tek tek test için)

YAML `clicks:` anahtarları ↔ Bukkit `ClickType`:

| YAML key | Oyuncu eylemi | Test önerisi |
|----------|---------------|--------------|
| `left` | Sol tık | Her demo butonda |
| `right` | Sağ tık | `demo-click-test` menüsü |
| `shift-left` | Shift + sol | aynı |
| `shift-right` | Shift + sağ | aynı |
| `middle` | Orta tık (creative) | creative modda |
| `drop` | Q | hover + Q |
| `control-drop` | Ctrl + Q | hover + Ctrl+Q |
| `swap-offhand` | F | hover + F |
| `number-key-1` … `number-key-9` | Hotbar 1–9 | hover + rakam |

### Built-in action türleri (0.1.6.3-alpha.1)

| Action | Açıklama | Demo menü |
|--------|----------|-----------|
| `noop` | Hiçbir şey | click-test |
| `close` | Menüyü kapat (normal) | pwarp layout |
| `refresh` | Mevcut sayfayı yenile | pwarp layout |
| `page-prev` | −1 sayfa | pwarp layout |
| `page-next` | +1 sayfa | pwarp layout |
| `sort-cycle` | Sıralama döngüsü (VISITS→NAME→RECENT) | pwarp layout |
| `filter-all` | Filtre: tümü | pwarp layout |
| `filter-favorites` | Filtre: favoriler (demo) | pwarp layout |
| `filter-mine` | Filtre: benimkiler (demo) | pwarp layout |
| `open-menu:<id>` | Başka YAML menüsü aç | hopper → dropper |
| `content-click` | ContentProvider handler | pwarp kafası tık |

Parametreli örnekler:

```
open-menu:demo-hopper
open-menu:demo-dropper
open-menu:demo-overlay-locked
```

### Mandatory özel

| Action / olay | Açıklama |
|---------------|----------|
| `afk-melon` | Karpuz seçildi → `release(SYSTEM)` |
| ESC | Menü kapanır → anında reopen |
| `/mngui release` | ADMIN release + log |
| `/mn reload` | Kapanır → reload bitince reopen |
| Restart | Mandatory session silinir — kaçış |
| Relog (`escape-on-quit: false`) | Join'de reopen |
| Relog (`escape-on-quit: true`) | Kaçış |
| Ölüm (`escape-on-death: false`) | Respawn'da reopen |

### Placeholder (title / lore)

| Placeholder | Anlam |
|-------------|--------|
| `{page}` | Mevcut sayfa |
| `{max_page}` | Toplam sayfa |
| `{target_page}` | Prev/next hedef sayfa |
| `{pages_back}` | Geride kalan sayfa sayısı |
| `{pages_forward}` | İleride kalan sayfa sayısı |

Stack cap 64 olsa bile lore `{target_page}` gerçek sayfayı gösterir.
